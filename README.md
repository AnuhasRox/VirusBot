# VirusBot - Virus Generator Framework

## Installation on Kali Linux

```
apt-get install python3-pip
git clone 
cd VirusBot
python3 -m pip install -r requirements.txt
python3 VirusBot.py
```

## Installation on Ubuntu 20.04

```
sudo apt-get install git python3-pip
git clone 
cd routersploit
python3 -m pip install -r requirements.txt
python3 VirusBot.py
